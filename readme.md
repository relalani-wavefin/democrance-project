#Democrance Test Application

### Prerequisites

```
Install Maven and  JAVA JDK 8
```
## Running the code

```
mvn spring-boot:run
```


## Running the test suite

```
mvn clean test
```

## Swagger URL for API documentation 

```
http://localhost:54274/swagger-ui.html
```

## Built With

* [Maven](https://maven.apache.org/) - Dependency Management
